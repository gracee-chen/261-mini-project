\begin{abstract}
We benchmark five image classification methods on Caltech-101: four deep learning architectures (ResNet-50, EfficientNet-B2, Vision Transformer-B/16, ConvNeXt-Tiny) and one classical baseline (SVM with ResNet-18 features). All deep models use ImageNet-pretrained weights and are fine-tuned under consistent settings; the SVM extracts frozen CNN features before training. Evaluation reports accuracy, per-class recall, and F1-scores on held-out test data. Our implementation is open-source and reproducible. Code and trained models are available at \url{https://github.com/gracee-chen/261-mini-project}.
\end{abstract}
